  • ReGi-PPI

    • It is a Deep Learning Model which predicts if a group of proteins interact with each other and form a network. The model was trained where the embeddings of each protein's CLS tokens were used as node features and the edges were based on whether the proteins interacted or not. The embeddings which were used as node features were extracted using the ProtBert Transformer. A graph of a PPI-network was formed which was passed through a Graph Neural Network Model which was trained to give predictions. A journal is to be published on it.

  • Face Patrol

    • It is a webapplication which basically detects an interuder within an organisation. We used state of the art model for object identfication which is YOLO-V8, which we trained based on our dataset which we collected. The admin can add users and cameras which are wire connected or wirelessly connected. If an intruder is detected it will send a signal directly to the logs and alert the system.
